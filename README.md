# Time Spent on Digital

Este projeto rastreia o tempo gasto em diferentes atividades digitais.

## Instalação

1. Clone o repositório: `git clone https://github.com/Ismael-Alfredo/time-spent-on-digital.git`
2. Navegue até a pasta do projeto: `cd time-spent-on-digital`
3. Instale as dependências (se houver): `pip install -r requirements.txt`

## Uso

Execute o script principal: `python main.py`
